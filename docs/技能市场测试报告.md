## 已知问题

### 1. LLM 翻译功能不可用 ⚠️

**问题描述**: LLM 网关返回空响应（choices 为空）

**影响范围**: 翻译服务无法自动翻译技能名称和描述

**降级方案**: 翻译失败时返回原文，不影响系统运行

**解决方案**:
1. 检查 LLM 网关配置（http://127.0.0.1:3180）
2. 验证 `LLM_API_KEY` 是否有效
3. 确认 `TRANSLATION_MODEL` (gpt-5.4-mini) 是否可用
4. 检查 new-api 后台日志

### 2. 测试数据版本信息缺失

**问题描述**: 技能详情 API 返回的版本列表为空

**原因**: 测试数据脚本未创建版本记录

**影响**: 无法测试版本相关功能

**解决方案**: 更新测试数据脚本，添加版本记录
