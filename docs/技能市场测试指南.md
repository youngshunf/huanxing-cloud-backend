## 故障排查

### 1. 翻译服务失败

**问题**：翻译返回原文
**原因**：LLM 网关未运行或配置错误
**解决**：
1. 检查 LLM_API_BASE_URL 配置
2. 确认 LLM 网关运行在 http://127.0.0.1:3180
3. 检查 new-api 或其他 LLM 网关状态

### 2. 本地打包失败

**问题**：下载返回错误
**原因**：repo_path 不存在或路径错误
**解决**：
1. 检查 HUANXING_HUB_LOCAL_PATH 配置
2. 确认技能的 repo_path 字段正确
3. 验证文件存在：`ls -la $HUANXING_HUB_LOCAL_PATH/skills/productivity/email-writer/`

### 3. 下载记录未保存

**问题**：数据库中没有下载记录
**原因**：事务未提交
**解决**：
1. 检查代码中是否有 `await db.commit()`
2. 查看数据库日志
