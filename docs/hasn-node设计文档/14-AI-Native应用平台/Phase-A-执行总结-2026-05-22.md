## 🔍 发现的问题

### 问题 1: 登录接口参数不匹配 ⚠️
**现象**: 
```json
{
  "code": 422,
  "msg": "请求参数非法: username 字段为必填项"
}
```

**原因**: 
- 测试脚本使用 `phone` + `code` (验证码登录)
- 实际接口需要 `username` + `password` (密码登录)

**影响**: 无法完成登录，后续测试无法进行

**解决方案**:
1. 修改测试脚本使用正确的登录参数
2. 或者找到正确的验证码登录接口

### 问题 2: Backend health 端点 404
**现象**: `GET /health` 返回 404

**原因**: health 端点可能在不同路径

**解决方案**: 
- 使用 `/docs` 查看 API 文档
- 或者跳过 health 检查，直接测试业务接口

**状态**: 因登录失败而阻塞

**影响**: 无法验证 Agent JWT 生命周期
