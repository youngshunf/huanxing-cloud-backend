### Phase 2: 实时通信（hasn-node）

#### 2.1 WebSocket 事件推送方法（4个）
在 `crates/hasn-node/src/surface/ws_dispatcher.rs` 中实现：

1. **push_session_message_received** - 推送完整消息接收事件
2. **push_session_message_chunk** - 推送流式消息分块
3. **push_session_status_changed** - 推送 Session 状态变更
4. **push_session_event** - 推送 Session 执行事件

#### 2.3 流式响应接口
在 `crates/hasn-runtime-adapter/src/adapter.rs` 中定义：

- **MessageChunk** 结构体 - 流式消息分块数据结构
- **execute_task_streaming** 方法 - RuntimeAdapter trait 流式执行接口

**设计理念**：
- 支持不同 runtime 实现流式响应
- 通过回调函数传递消息分块
- 保持与现有 dispatch 方法的兼容性
